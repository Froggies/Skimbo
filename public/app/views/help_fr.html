Généralités

Skimbo rassemble de nombreux réseaux-sociaux dans une seule fenêtre de votre navigateur. Pas besoin de rafraichir votre navigateur ! Lorsque de nouveaux messages sont disponibles, Skimbo les affiche pour vous en temps réel dans la fenêtre.

Providers - services

Dans le jargon de Skimbo, les réseaux-sociaux sont des providers. Chaque provider offre plusieurs services. Typiquement, le provider Facebook, offre le service wall, ce sont tous les messages qui vous pouvez voir sur votre mur facebook. Il offre également le service messages personnels, qui sont les messages privés envoyés à un utilisateur ou qu'un utilisateur vous a envoyé. Voici la liste des providers et de leurs services :

Mon compte

Votre compte rassemble toutes les données vous concernant, que Skimbo a récupéré auprès des providers dont vous lui avez donné accès.

Où le trouver ?

Votre compte est accessible par le bouton tout en haut du menu à gauche. Il vous suffit de cliquer sur l'image d'un de vos comptes relié à Skimbo.

Mettre en pause un accès

Pour que Skimbo arrête de récupérer vos données d'un ou plusieurs providers, jusqu'à votre prochaine connexion, il vous suffit de cliquer sur le symbole pause (2 traits verticaux), dans votre compte.

Supprimer un accès

Si vous souhaitez que Skimbo ne récupère plus de données sur un ou plusieurs providers, vous pouvez tout simplement lui retirer son droit d'accès. Pour cela, rendez-vous dans votre compte et cliquez sur la croix en face du provider. Attention, Skimbo ne sera plus en mesure de retrouver vos colonnes, si après avoir supprimer un accès, vous vous re-connectez par le biais de ce provider. En effet, en toute transparence, Skimbo supprime réellement ce droit d'accès et toutes les données (nom, prénom, image...) qui y sont reliés. Si vous avez des colonnes contenant des services de ce provider, vous devrez les supprimer si vous ne voulez pas que Skimbo affiche "Vous êtes déconnecté de..." à chaque connexion.

Déconnexion

Pour vous déconnecter de Skimbo, allez dans votre compte et cliquez sur "déconnexion". Vous serez alors redirigez vers la page de login, et Skimbo arrêtera d'aller chercher des nouveaux messages chez les différents providers. Vous devrez vous reconnecter avec un réseau se trouvant dans la liste des providers, pour que Skimbo retrouve vos colonnes.

Nous contacter

Pour nous contacter, il vous suffit de vous rendre dans votre compte et de cliquer sur le bouton "@ Nous contacter". Remplissez tous les champs et cliquer sur "Envoyer". Si le message ne nous parvient pas, vous serez prévenu par une notification.

Colonne

Les colonnes de Skimbo vous permettent de définir votre propre interface. Ces colonnes afficheront vos messages des providers de la manière dont vous le souhaitez. Pour cela ajouter des services dans ces colonnes, redimensionner les et enfin déplacer les comme vous le souhaitez.

Ajouter

Pour ajouter une colonne, il suffit de cliquer sur le bouton en forme de colonne avec un plus, dans le menu à gauche. Ensuite, choisissez un provider et un ou plusieurs services. Vous pouvez mixer les providers et leurs services comme vous le souhaitez. Une fois validé votre nouvelle colonne est disponible dans Skimbo. A tout moment vous pouvez la modifier, la déplacer, la redimensionner ou encore la supprimer.

Modifier

Cliquer sur le bouton en forme de roue en haut à droite de la colonne. Vous serez redirigé vers la même fenêtre que pour ajouter une colonne.

Supprimer

Cliquer sur "modifier" la colonne, puis le bouton "Supprimer" en bas de la fenêtre.

Redimensionner

Cliquer sur le bouton avec des flêches en haut à droite de la colonne. Vous pouvez cliquer sur un carré gris pour donner une taille à la colonne. Plus le carré est en bas à droite plus la colonne sera grand.

Déplacer

Cliquer sur le titre de la colonne, sans relacher. Puis déplacer la souris sur une autre colonne, pour que la 1ere prenne la place de la 2eme.

Messages

Lorsque Skimbo récupère un nouveau message il l'ajoute en haut de la colonne correspondante au provider et au service que vous avez choisis. De plus, il vous signal, que vous avez un (des) message(s) non lu par le petit chiffre rouge à côté de l'oeuil en haut à droite de la colonne. Lorsque vous remontez l'ascenseur, les messages sont automatiquement marqué comme lu. Visuellement ils passent de gris à blanc afin de savoir ce qu'on a lu ou pas.

Tous les messages lus

Vous pouvez marquer tous les messages comme lu en cliquant sur l'oeuil en haut à droite de la colonne.

Commenter

Lorsque le provider et le service du message le permettent, vous pouvez cliquer sur la flèche en bas à droite du message pour le commenter. Une fois fini tapper sur la touche "entrer" pour envoyer ce message.

Partager

Si vous cliquez sur le trident retourné, vous pourrez skimber le message.

Rafraichir

Cliquez sur la double flêche dessous le message pour le rafraichir.

Marquer

"Marquer" ou encore "Star" le message dans le jargon Skimbo, permet, en fonction du provider, de le retweeter, liker, thanker...

Médias

Lorsque le message contient un média (uniquement photo pour le moment), vous pouvez cliquer dessus pour l'agrandir.

Skimber

Skimber, signifie, dans le jargon de Skimbo, envoyer un message à tous les réseaux sociaux dont vous avez donné l'accès. Ou en d'autres termes, il s'agit d'inverser le flux de données. En effet, Skimbo permet de récupérer tous les messages des providers mais aussi d'envoyer un message sur tous ces providers.

Règles proviers

Skimbo vous permet d'envoyer un message sur plusieurs providers en même temps. Mais chaque provider à ses propres règles pour définir un message. Certains ont besoin d'un titre, d'autres d'un lien ou encore d'une image.
Voici l'ensemble des règles :
LogoProviderTitreLienImagedescription
Twitter Non utilisé Non utilisé Non utilisé Envoi un tweet sur votre mur. Toutes les règles de twitter restent les même (140 caractères max au delà le message est coupé, si votre message commence par @user ce message lui sera destiné...). Même si twitter accepte les liens et les images, ceux-ci doivent être directement dans le corps du message. Afin de ne pas entrer en conflit avec d'autres providers, nous avons préférés ne les pas intégrer directement pour vous.
Facebook Non utilisé Optionel Optionel Envoi un message sur votre mur facebook.
Facebook Non utilisé Optionel Optionel Envoi un message sur la page facebook choisie. Vous ne pouvez envoyer de message que sur les pages dont vous êtes administrateur.
Linkedin Obligatoire Optionel Optionel Envoi un message sur votre timeline Linkedin.
Github Obligatoire Non utilisé Non utilisé Créer un gist dans votre compte personnel, avec comme titre du fichier le titre et le corps est le message (WTF !? Oui le jour où j'ai dev ça, j'ai bien trippé).
Googleplus Obligatoire Optionel Optinel Créer un message dans Googleplus dans un endroit complètement caché. Seuls les membres de vos cercles et utilisant Skimbo pourraient y avoir accès. Nous n'avons jamais finit ce dev, en attendant qu'ils ouvrent leur api un petit plus.
Scoopit Obligatoir Optionel Optionel Créer un topic dans le board sélectionné. Vous devez être habilité à créer des topics dans ce board pour le sélectionner.
Viadeo Obligatoir Optionel Optionel Envoi un message sur votre timeline Viadeo.

Planifier

Vous pouvez également, planifier ce message, afin que Skimbo l'envoi pour vous à une date et uen heure précise. Pour cela, il vous suffit de cliquer sur "planifier" une fois votre message écrit. Puis sélectionnez une date et une heure, votre message sera alors envoyé au moment choisis, avec un maximum d'une minute d'écart. Attention, si entre temps, vous avez révoqué l'accès de Skimbo auprès du provider, ce message ne pourra pas être envoyé.
