SkimboU : Ubuntu touch client pour Skimbo

SkimboU est une application pour les prochains téléphones tournant sous UbuntuTouch. Il permettra d'accéder à toutes les fonctionnalités de Skimbo. Vous pouvez suivre son avancé sur Github. Il est entièrement codé en Qt 5.1.
