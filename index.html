Skimbo

This app was created during the Typesafe_ contest app with Play2_. Test it on Demo_

For the Typesafe contest, we made a web application Skimbo : a social networks mashup. The main problem that our application solves is that most user are members of several social networks so to follow every changes on each social network, users have to open one tab for each one, which is not very convenient. Thanks to our application, the user can follow all its social networks, in a unique window, thanks to a new kind of display.
