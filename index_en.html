Technologies

Toute l'équipe Froggies remercie ceux qui oeuvrent à faire du web un meilleur monde virtuel. En effet, Skimbo est bâti sur une pile technologique de dernière génération, open-source, gratuite et très performante. Au final, rien ne serait possible sans eux.

La TypeSafe stack permet à Skimbo d'utiliser au mieux les serveurs disponibles. Lorsque cela sera nécessaire, nous pourrons agrandir le nombre de serveurs sans aucun développement supplémentaire. Cette stack se décompose en 3 composants majeurs : le language de programmation Scala, le framework playframework et ce qui permettra à Skimbo de supporter le nombre d'utilisateurs : les acteurs Akka.

MongoDb est la base de donnée non sql qui contient toutes les données de Skimbo. Nous l'utilisons par le biai du driver scala ReactiveMongo. Ce driver permet de faire discuter Skimbo avec mongoDb.

Même si le plus gros travail de Skimbo est de remonter toutes vos informations des différents réseaux sociaux, il faut aussi qu'il vous les présente avec la forme. Pour cela, nous utilisons également les dernières technologies du web. HTML5, CSS3 et Javascript font partis des compétences mises en oeuvre pour Skimbo. AngularJs est ce qui permet de rendre le tout vivant et agréable à utiliser.

Dans un projet, et tout particulièrement un projet informatique, la communication est primordiale. Que cela soit pour discuter d'améliorations, d'ergonomie ou encore de bugs nous utilisons Github. Il nous permet de gérer le versionnage de notre code, mais aussi d'avoir un wiki et un bug tracker. Encore mieux il héberge également ce site web !

Enfin, pour que vous puissiez accéder à Skimbo, il faut qu'il soit hébergé sur un serveur. Après avoir essayé AmazonWebService, DotCloud et Heroku, nous nous sommes arrêté chez OpenShift.
