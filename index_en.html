Technologies

Froggies team thanks those who work to make the web a better virtual world. Indeed, Skimbo is built on a latest generation technology stack, open-source, free, and very effective. At the end of the day, nothing would be possible without them.

Typesafe stack allows Skimbo to use available servers to their maximum capabilities. When it will be necessary, we can scale without any efforts. This stack is composed by 3 majors components : Scala programation langage, playframework, and the tool which allow Skimbo to handle the load and concurrency issues : The Actor model with Akka

MongoDB is the database system which contains all Skimbo datas. We use it through ReactiveMongo scala driver. This driver allow the communication between Skimbo and MongoDB.

Event if the majort part of Skimbo job is to gather all your social network information, it must shows them with a shiny interface. To do that, we also use latest web technologies. HTML5, CSS3 and Javascript are part of the skills used for Skimbo. AngularJS is the glue which allow all of that to be alive and nice to use.

In a project, specially a software project, communication is king. Whether to discuss improvements, ergonomics, or even bugs we use Github. It allow us handle code versionning, wiki system, bug tracking and even better it hosts this web site !

Finally, to access Skimbo, it need to be host on a server. After trying AmazonWebService, DotCloud and Heroku, we stopped at OpenShift.
